Como instalar o plugin ?

  1. 1. No menu do Eclipse, selecione Help -> Install New Software...
  2. 2. Na janela Install, no campo Work with, digite http://143.107.58.177/baduino/
  3. 3. Marque a ultima versão da Baduino e clique em Next
  4. 4. Siga os passos do instalador.

Requisitos para coletar a cobertura

A Baduino parte do pressuposto que o projeto esta estruturado no padrão maven.
• Ter o plugin do Maven instalado no Eclipse,
• Classes compiladas estão localizadas em: [projectDir]/target/classes,
• Classes de testes compilados estão localizados em: [projectDir]/target/test-classes,
• Todas as classes de testes devem terminar com *Test.class

Se o seu projeto não está estruturado dessa forma, é necessário criar um arquivo auxiliar chamado baduino.properties que precisará conter 2 campos:

  • compiled.tests.dir=<localização dos pacotes de teste compilados>
    Exemplo no Windows: compiled.tests.dir=C:\\Users\\youruser\\yourproject\\bin\\test-classes
  • compiled.classes.dir=<localização das classes compiladas>
    Exemplo no Linux: compiled.classes.dir=/home/youruser/yourproject/bin/classes
    • OBS.: A barra invertida '\' é um caractere de escape em java para String, portanto, Usuários de windows devem usar dupla barra invertida '\\'.

Example courtesy Martin Bean and Ryan Fait.
