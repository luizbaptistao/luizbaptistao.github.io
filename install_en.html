  2. 2. Download the
  3. 3. Paste it into {eclipse_folder}\dropins\

Now, its possible to see method DUAs, but isn't possible collect DUAs coverage yet. If you want know how to use the plugin click here

What else do I need to collect DUAs coverage ?

  1. Default location: [projectDir]/jacocoagent.jar
  2. Default location: [projectDir]/jaguar.jar
  3. A folder containing the compiled classes. Default location: [projectDir]/target/classes
  4. A folder containing the compiled tests. Default location: [projectDir]/target/test-classes

What if they are somewhere else ?

If one of the previous files (jacocoagent.jar and jaguar.jar) and folders (classes and test-classes) are not in the default location you need to specify them in a file named baduino.properties. This file MUST be in the root of the project and follow this pattern.

  • jacoco.agent.jar=<jacocoagent.jar location>
  • jaguar.jar=<jaguar.jar location>
  • compiled.tests.dir=<Compiled tests location>
  • compiled.classes.dir=<Compiled class location>
  • project.dir=<Root location>

Note: Every class test must be ended with *Test.class
