How to install ?

  1. 1. From your Eclipse menu select Help → Install New Software...
  2. 2. In the Install dialog enter http://143.107.58.177/baduino/ at the Work with field.
  3. 3. Check the latest EclEmma version and press Next
  4. 4. Follow the steps in the installation wizard.

Requirements to collect coverage.

Baduino assumes that project follow the default maven archetype. A Baduino parte do pressuposto que o projeto esta estruturado no padrão maven.
• Maven plugin installed on Eclipse,
• Ter o plugin do Maven instalado no Eclipse,
• Compiled Classes located in: [projectDir]/target/classes,
• Classes compiladas estão localizadas em: [projectDir]/target/classes,
• Complied Test Classes located in : [projectDir]/target/test-classes,
• Classes de testes compilados estão localizados em: [projectDir]/target/test-classes,
• Every test class must be ended with *Test.class
• Todas as classes de testes devem terminar com *Test.class

If your project was not structure on this way, its necessary create an archive called baduino.properties that will contain two fields:

Se o seu projeto não está estruturado dessa forma, é necessário criar um arquivo auxiliar chamado baduino.properties que precisará conter 2 campos:

  • compiled.tests.dir=<compiled package test class location>
    Windows sample: compiled.tests.dir=C:\\Users\\youruser\\yourproject\\bin\\test-classes
  • compiled.classes.dir=<compiled class locations>
    Linux sample: compiled.classes.dir=/home/youruser/yourproject/bin/classes
    • Note: The backslash '\' is a Java escape character for String and char literals, so, Windows users must use double backslash '\\' to escape it.
